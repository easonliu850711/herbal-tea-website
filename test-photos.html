🌸 劉媽媽の草本茶 - 照片測試頁面

測試說明：

這個頁面直接測試圖片檔案是否能正確載入。如果圖片顯示正常，表示檔案路徑正確。

如果圖片無法顯示，可能是路徑問題或檔案不存在。

解決方案：

  1. 使用 Next.js 開發伺服器（推薦）： 
    cd herbal-tea-website
npm install
npm run dev
    然後訪問 http://localhost:3000
  2. 修正靜態路徑： 如果要在靜態 HTML 中查看，需要將路徑改為相對路徑
  3. 使用正確的圖片引用方式： Next.js 使用 <Image> 組件或 CSS background-image 時，路徑是相對於 public 目錄的
