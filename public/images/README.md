## 🎯 照片使用指南

### 網站整合：
1. **首頁展示**：精選照片輪播
2. **照片畫廊**：完整分類瀏覽
3. **品牌故事**：關鍵照片插圖
4. **產品頁面**：產品相關照片

### 情感設計：
1. **真實性**：展示市場真實場景
2. **溫暖感**：強調人情味和互動
3. **故事性**：每張照片都有故事
4. **專業性**：呈現品質和用心

## 📋 檔案命名規範

### 建議格式：
```
分類-主題-日期-序號.jpg
```

### 範例：
```
stall-environment-morning-20260323-01.jpg
product-closeup-tealeaves-20260323-01.jpg

```

## 🔧 技術規格

### 支援格式：
- JPG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

### 建議尺寸：
- 英雄區背景：1920×1080px
- 產品主圖：800×800px
- 故事照片：1200×800px
- 頭像照片：400×400px

### 檔案大小：
- 建議每張 < 5MB
- 最大支援 10MB
