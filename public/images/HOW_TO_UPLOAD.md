# 📸 照片上傳指南

## 快速開始：
1. 將照片檔案複製到：`public/images/temp-upload/`
2. 運行處理腳本：`./scripts/process_photos.sh`
3. 查看結果：照片會自動分類並更新網站

## 支援格式：
- JPG/JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp)

## 自動分類規則：
根據檔案名關鍵字自動分類：

| 關鍵字 | 分類 | 說明 |
|--------|------|------|
| stall, market, 攤位 | 攤位環境 | 市場攤位照片 |
| poster, 海報, japan, 日本 | 行銷素材 | 宣傳海報材料 |
| product, 產品, tea, 茶 | 產品特寫 | 茶葉產品照片 |
| interaction, 互動, customer, 顧客 | 互動場景 | 顧客互動照片 |
| process, 製作, making, 包裝 | 製作過程 | 生產製作照片 |

## 處理效果：
1. ✅ 自動分類到對應目錄
2. ✅ 生成優化檔案名（時間戳+序號）
3. ✅ 創建 JSON 元數據檔案
4. ✅ 情感價值分析（真實性、溫暖度、故事性、專業性）
5. ✅ 網站畫廊自動更新
