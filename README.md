# 劉媽媽的草本茶 - 官方網站

## 🎯 專案概述
劉媽媽的草本茶官方網站，展示三十年市場經驗的草本養生茶產品。

## 🚀 技術棧
- **框架**: Next.js 14 (App Router)
- **樣式**: Tailwind CSS + 自定義設計系統
- **動畫**: Framer Motion
- **字體**: Inter + 思源黑體/宋體

## 🎨 設計特色
1. **情感化設計**: 圍繞劉媽媽三十年市場經驗的溫暖品牌故事
2. **QR Code 整合**: 完整的 Line 官方帳號掃碼界面
3. **響應式設計**: 完整支援桌面、平板、手機裝置
4. **性能優化**: 圖片優化、字體子集化、代碼分割

## 📁 專案結構
```
herbal-tea-website/
├── src/
│   ├── app/           # Next.js App Router
│   ├── components/    # React 組件
│   └── lib/          # 工具函數
├── public/           # 靜態資源
└── config/          # 配置文件
```

## 🚀 快速開始
```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 生產構建
npm run build
npm start
```
